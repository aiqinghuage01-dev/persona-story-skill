# 人设故事模板结构（员工视角讲老板）

## 核心公式

> 先让你心疼这个人 → 再让你佩服这个人 → 最后让你买这个人的东西

## 七段结构

### 1. 钩子（约占5%）

**功能**：3秒建立好奇
**公式**：人物标签 + 反常识数字/特质 + 悬念预告
**要求**：
- 必须有一个具体的身份标签（"我的老板是厂二代""我们教练教了十八年乒乓球"）
- 必须有一个让人想继续听的反差（亏七百万、清华毕业去教拍视频、劝客人少来）
- "最近发生了一件事"作为悬念过渡句

### 2. 冲突（约占15%）

**功能**：情绪过山车，从"好事"瞬间跌到"搞什么"
**公式**：一件具体的事 + 预期很好 → 结果反转
**要求**：
- 必须是一件具体的、可以画面化的事件（不是抽象感悟）
- 情绪从正面到负面的落差要大
- 用员工的吐槽视角拉近距离（"我说老板你疯了？"）

### 3. 翻转（约占20%）

**功能**：给观众认知增量，提升完播率
**公式**："原来…" / "后来我才明白…" + 行业真相/供应链知识
**要求**：
- 必须是观众之前不知道的信息（不是常识）
- 要讲清楚一条因果链（A导致B导致C导致D）
- 让观众觉得"学到了"

### 4. 弧光（约占15%）

**功能**：人物的决策揭示价值观，情感高潮
**公式**：老板做了一个决策 + 背后的真实动机 + 一句灵魂金句
**要求**：
- 先给行为（他怎么做的），再给动机（他为什么这么做）
- 金句要把"看似傻的行为"翻转成"值得敬佩的坚持"
- 这句话是全篇最值得被截图传播的一句

### 5. 接杠（约占15%）

**功能**：主动接住评论区可能的质疑
**公式**：引用质疑 + 用调侃/反问回击 + 用事实堵嘴
**要求**：
- 先把杠精的话说出来（"有人看不懂""有人说你…"）
- 不defensive，用幽默化解（"装能装出七百万，我也想学"）
- 紧接着用成本账/事实回击

### 6. 成本账（约占15%）

**功能**：让观众理解老板扛着的真实压力
**公式**：逐项列举成本 + "他扛着的不是X，是Y"
**要求**：
- 至少列3-5项具体成本（仓储、人工、设备、原料、房租等）
- 每一项都要有画面感（"工人站着就得发工资""睁眼就是钱"）
- 收束句式："他扛着的不是一件/一锅/一米XXX赚几块，是XX年来XXX这件事"

### 7. 产品+收口（约占15%）

**功能**：自然过渡到产品介绍，用人设做收口
**公式**："说回…" + 产品核心卖点（只讲成本和功能）+ 人设收口金句
**要求**：
- 过渡句用"说回他的XXX"或"说回他到底在卖什么"
- 产品描述只讲材质、工艺、功能，不讲"值不值"
- 价格一笔带过，不做促销
- 最后一句用人设做收口，风格统一为"XX年了，XXX。习惯就好。"
