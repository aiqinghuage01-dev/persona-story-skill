# 人设故事 Skill 质量测试用例

## 测试方法

用以下测试输入跑skill，检查输出是否符合标准。

---

## 测试用例1：餐饮老板

**输入**：
- 知识库：重庆火锅店，开了8年，主打牛油锅底现炒
- 故事素材："上个月有个顾客要求我们把锅底换成清油的，说牛油太腻了。我说我们只做牛油，他说那我去别家了。"

**检查项**：
- [ ] 开头有具体身份标签和反常识点
- [ ] 冲突围绕"拒绝顾客"展开
- [ ] 翻转段讲了火锅行业的锅底真相（牛油vs清油成本/工艺差异）
- [ ] 有灵魂金句
- [ ] 产品段介绍了牛油锅底的工艺/原料
- [ ] 收口风格统一（"习惯就好"式）
- [ ] 600-900字
- [ ] 无AI味

---

## 测试用例2：教培老板

**输入**：
- 知识库：钢琴培训工作室，开了6年，主教4-12岁小朋友
- 故事素材："有个家长花了三万报了年卡，孩子学了两个月不想来了。家长要退费，我退了。同行都说我傻。"

**检查项**：
- [ ] 开头有具体身份标签和反常识点
- [ ] 冲突围绕"退费"展开
- [ ] 翻转段讲了少儿培训行业退费/续费的行业真相
- [ ] 有灵魂金句
- [ ] 产品段介绍了课程形式/教学方式
- [ ] 600-900字
- [ ] 无AI味

---

## 测试用例3：制造业老板

**输入**：
- 知识库：纸杯纸碗生产厂，干了15年，食品级原纸
- 故事素材："一个大客户要求我们把纸的克重降低，这样成本能省一半。我没答应。"

**检查项**：
- [ ] 开头有具体身份标签和反常识点
- [ ] 冲突围绕"拒绝大客户降低品质"展开
- [ ] 翻转段讲了纸杯行业克重/原纸的行业真相
- [ ] 有灵魂金句
- [ ] 产品段介绍了食品级原纸的工艺
- [ ] 600-900字
- [ ] 无AI味

---

## 通用检查清单

每条输出都必须通过：

| # | 检查项 | 通过标准 |
|---|---|---|
| 1 | 视角一致 | 全篇员工第一人称，没有跳出 |
| 2 | 七段完整 | 钩子、冲突、翻转、弧光、接杠、成本账、产品收口全有 |
| 3 | 无AI味 | 没有"在当今社会""值得注意的是""随着…的发展" |
| 4 | 无硬推销 | 没有"赶紧下单""限时优惠""最后X个名额" |
| 5 | 有认知增量 | 观众能从翻转段学到一个行业知识点 |
| 6 | 有灵魂金句 | 至少一句值得截图传播的话 |
| 7 | 口语化 | 能直接读稿，不拗口 |
| 8 | 字数合规 | 600-900字 |
| 9 | 故事真实 | 核心故事来自用户提供的素材，不是编的 |
| 10 | 收口统一 | 最后一句是温暖幽默的人设收口 |
